Устройство

Git — контентно-адресуемая файловая система

WTF?

Все — файл: коммиты, стуктура папок, ветки

Название: Хеш содержимого (контента)

$ echo 'test content' | git hash-object --stdin -w d670460b4b4aece5915caf5c68d12f560a9fe3e4

Хранилище ключ-значение

Папка .git/objects

$ find .git/objects -type f .git/objects/d6/70460b4b4aece5915caf5c68d12f560a9fe3e4

Магия

$ git cat-file -p d670460b4b4aece5915caf5c68d12f560a9fe3e4 test content
